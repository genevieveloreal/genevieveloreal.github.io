Gamify

Gamify is a Gamification Marketing leader that helps companies make fun and enaging ‘AdverGames’. After recently partnering with Wix, Gamify identified the need for a website refresh, including the development of a Learning Centre to help educate potential customers about the benefits of gamification.

I developed main pages for this project including the Home page, Partner page, Direct page, Wix page and Learning Centre. The client wanted to be able to display pricing on the Direct page based on the user’s region, while also allowing them the option to change the currency. I created this functionality using HubSpot’s smart content features in conjunction with jQuery and AJAX to get conversion rates from a currency API.
